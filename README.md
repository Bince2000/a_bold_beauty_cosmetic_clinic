# Ambarsariya Beauty Parlour — Frontend App

React + Tailwind CSS + DaisyUI  
Theme: Lavender & White

## Getting Started

```bash
# 1. Install dependencies
npm install

# 2. Start the dev server
npm run dev
```

Then open http://localhost:5173 in your browser.

## Pages

- **Home** — Hero, features, service previews, testimonials
- **Services** — Full service list with category filter tabs
- **Gallery** — Masonry photo grid with lightbox
- **Book Appointment** — Validated booking form with confirmation screen

## Build for production

```bash
npm run build
npm run preview
```
